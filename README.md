# go-web-demo

A demo website written in Go.

## Usage

1. Ensure <project_dir> in $GOPATH

2. Entry to <project_basedir>

```bash
cd <project_basedir>
```

3. Change mode or script:

```bash
chmod +x deploy.sh
```

2. Run it:

```bash
./deploy.sh
```
