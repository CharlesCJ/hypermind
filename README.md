# go-web-demo

A demo website written in Go.

## Usage

1. Build project with a tool of Go:

```bash
cd <project_basedir>
go build server
```

2. Run it:

```bash
chomd +rx ./server
./server
```

Or, run it directly:

```bash
go run <project_basedir>/src/go-web-demo.go
```
